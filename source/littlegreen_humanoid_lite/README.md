# Littlegreen Humanoid Lite Training Package

This package contains the Isaac Lab task definitions and RSL-RL training configuration for Littlegreen-Humanoid-Lite v2.0.0. The v2.0.0 release is a behavior-preserving rename of the Berkeley-Humanoid-Lite v1.4.8 Track 1 source.

Public `Velocity-Lilgreen-*` task IDs and the two legacy Berkeley task IDs are intentionally preserved.
